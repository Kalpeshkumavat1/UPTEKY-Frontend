# Frontend - Feedback Management Dashboard

React.js frontend built with Vite for the Feedback Management Dashboard.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
VITE_API_URL=http://localhost:5000/api
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Pages

- `/` - Feedback submission form
- `/dashboard` - Analytics dashboard with feedback table
